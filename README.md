## レシピアプリ
https://my-recipe-yuki-s2s-projects.vercel.app/

https://github.com/yuki-s2/my-recipe

## 使用技術
言語: TypeScript, Next.js, React.js, JavaScript, HTML, CSS
その他: Firebase, Vercel

## 主な機能
- Firebaseを使用したログイン機能
- レシピの名前、画像、作り方を登録
- 登録したレシピの編集、削除
- 登録したレシピから今日の献立をピックアップする機能
